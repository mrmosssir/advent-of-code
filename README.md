# Advent of Code

This repository contains solutions for Advent of Code challenges.

## Structure

```
2025/
├── day1_solution/    # Day 1 solution
├── day2_solution/    # Day 2 solution
└── ...
```

## Running the Solutions

Each day's folder is an independent Rust project and can be run separately:

### Day 1
```bash
cd 2025/day1_solution
cargo run
```

### Day 2
```bash
cd 2025/day2_solution
cargo run
```

## Requirements

- Rust (latest stable version recommended)
- Cargo

## Installing Rust

If you haven't installed Rust yet, you can install it with:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Input Files

Each solution requires its corresponding input file (`input.txt` or specific filename). Please place the input data from the Advent of Code website into the appropriate file.

## About Advent of Code

[Advent of Code](https://adventofcode.com/) is an annual programming challenge event that provides 25 days of puzzles each December.
